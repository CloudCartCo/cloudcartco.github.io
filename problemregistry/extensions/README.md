# Global Extension Fields

This section contains the global extension fields that may be included in a Problem Details response associated with any problem type in this registry. 

Extension fields are used to provide additional context for the error and are described in the [Extension Members](https://www.rfc-editor.org/rfc/rfc9457.html#name-extension-members) section of the Problem Details RFC.

### Registered Extensions
* [traceId](./traceId.md): A statistically unique identifier generated on the server to correlate a specific error response with server-side logs and telemetry.
