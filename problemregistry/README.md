# Problem Registry

This registry defines the Problem Types used by our APIs to communicate various errors and conditions to callers. All definitions in this registry are compliant with [IETF Problem Details RFC 9457](https://www.rfc-editor.org/rfc/rfc9457.html).

### Registry Navigation
* [Problem Types](./problems/) — A catalog of specific error categories (e.g., validation errors).
* [Global Extensions](./extensions/) — Supplemental fields that may appear in a problem response for any problem type for added context.
