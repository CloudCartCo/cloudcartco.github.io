# Problem Types

This section contains a catalog of specific Problem Types compliant with the [IETF Problem Details RFC 9457](https://www.rfc-editor.org/rfc/rfc9457.html). These types provide machine-readable and human-readable documentation for API errors.

### Catalog
* [Validation Error](./validation-error.md): Used when a request is syntactically correct but fails semantic or business rule validation (Status 422).
